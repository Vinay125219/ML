üíª Manual Commands

                
Copy and paste these commands in your terminal:

                
                
Check Retraining Status:

                
curl http://localhost:8002/status

                
                
Trigger Retraining (All Models):

                
curl -X POST "http://localhost:8002/retrain" -H "Content-Type: application/json" -d '{"force": true}'

                
                
Retrain Specific Model:

                
curl -X POST "http://localhost:8002/retrain" -H "Content-Type: application/json" -d '{"model_type": "housing", "force": true}'

                
                
View Results:

                
curl http://localhost:8002/results
